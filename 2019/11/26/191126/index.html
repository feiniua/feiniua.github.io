常见命令

cd 【】 进入到指定目录(./当前目录 ../上一级 /根目录)
ls 查看目录信息
pwd 查看当前路径
adduser 【】 添加用户
userdel 【】 删除用户
groupadd 【】 添加组
groupdel 【】 删除组
userdel -r 【】 把所在组信息也删除
mkdir 【】 新建目录
rmdir 【】 删除目录
touch 【】 新建文件
cp 【源文件】 【目标文件】 复制
cp -r【】【】 递归复制整个文件夹
rm 【】 删除文件
rm -r 【】 递归删除文件夹
rm -f 【】 无提示
rm -rf 【】
mv 【】 【】 移动
cat 【】 以只读方式打开文件
cat -n 【】 显示行号
more 【】 cat优化
less 【】 查看大文件
【】 > 【】 将输出的信息覆盖入后面的文件中
【】 >> 【】 追加到文件的末尾
cat 文件1 > 文件2 将文件1的内容覆盖到文件2中
echo 【选项】 【输出内容】
head 【】 用于显示文件的开头部分，默认10行
head -n 5 【】 输出开头5行
tail 【】 查看文件尾部信息
tail -n 5 【】 输出尾部5行
tail -f 【】 实时追踪文档的所有更新
ls -s 【原文件或者目录】【软链接名】 存放了链接其它文件的路径
hostory 查看历史命令
hostory 【行数】
!【历史命令的行数】 执行历史命令
date 查看时间
date “+%Y %m %d %H %M %S”
date -s “【2018-10-4 10:20:22】” 设置系统当前时间
cal 查看日历
cal 【年份】 查看某年的日历
find 【搜索范围】【选项】 ( find /home -name 1.txt ) -name -user -size(+n大于)
grep 【选项】 【查找内容】 过滤查找 | 表示把前一个命令的处理结果输出传递给后面的命令处理
gzip 【文件】 压缩文件
gzip -r 【目录】
gunzip 【文件】.gz 解压缩文件
zip 【生成的zip文件名】 【需压缩的文件】
unzip -d 【指定的解压目录】 【需解压的文件】 若无目标目录会自动生成
tar -cvf 【压缩后的文件名】 【需要压缩的文件】 -c 生成压缩档案 -v 显示执行流程 -f 表示后面紧跟档案名字
tar -xvf 【需解压的文件】 -C 【解压文件放的目录】
压缩
tar –cvf jpg.tar *.jpg 将目录里所有jpg文件打包成tar.jpg
tar –czf jpg.tar.gz *.jpg 将目录里所有jpg文件打包成jpg.tar后，并且将其用gzip压缩，生成一个gzip压缩过的包，命名为jpg.tar.gz
tar –cjf jpg.tar.bz2 *.jpg 将目录里所有jpg文件打包成jpg.tar后，并且将其用bzip2压缩，生成一个bzip2压缩过的包，命名为jpg.tar.bz2
tar –cZf jpg.tar.Z *.jpg 将目录里所有jpg文件打包成jpg.tar后，并且将其用compress压缩，生成一个umcompress压缩过的包，命名为jpg.tar.Z
rar a jpg.rar *.jpg rar格式的压缩，需要先下载rar for linux
zip jpg.zip *.jpg zip格式的压缩，需要先下载zip for linux
解压
tar –xvf file.tar 解压 tar包
tar -xzvf file.tar.gz 解压tar.gz
tar -xjvf file.tar.bz2 解压 tar.bz2
tar –xZvf file.tar.Z 解压tar.Z
unrar e file.rar 解压rar
unzip file.zip 解压zip
