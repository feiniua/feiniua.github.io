final

1.final修饰的类不能被继承
2.final修饰的方法不能被重写
3.final修饰的变量不能被修改，并且只能显式赋值 final String s = "String";

abstract(对事物抽象)

有一个方法是抽象的，类就要声明成抽象类

interface(抽象类是对一类事物的高度抽象，其中既有属性也有方法。接口是对方法的抽象，也就是对一系列动作的抽象)

1.从本质上看，接口是一种特殊的抽象类，这种抽象类只包含常量和方法的定义，而没有实现
2.接口中所有的成员变量都默认是由public static final修饰的
3.接口中的所有方法都默认是由public abstract修饰的
4.接口采用多继承
5.如果一个类没有实现接口的方法就应该定义为抽象类
6.继承抽象类时如果需要新增抽象方法时使用接口就不需要修改子类
