一个Java文件中只能有一个public类

1.可以有多个类但只能有一个public类，这表示，每个编译单元都有单一的公共接口，用public类来表现。该接口可以按要求包含众多的支持包访问权限的类
2.当Java源文件中有多个非public类时，运行时需要对数据来源进行选择
3.编译器在编译时，针对一个java源代码文件（也称为“编译单元”）只会接受一个public类。负责报错
4.public类不是必须的，Java文件中可以没有public类
5.public类名跟文件名相同是为了方便虚拟机在相应的路径中找到相应的类所对应的字节码文件

简单类对象的实例化过程

1.方法区 加载Person.class
2.栈内存 在栈中申请空间，声明变量p
3.堆内存 在堆内存中开辟空间，分配地址。
4.堆内存 并在对象空间中，对对象中的属性进行默认初始化。
5.栈内存 构造函数方法进栈，进行初始化
6.初始化完毕后，将堆内存的地址赋给引用变量。构造方法出栈

子类对象的实例化过程

1.方法区 先加载Person.class,再加载Student.class
2.栈内存 在栈中申请空间，声明变量stu
3.堆内存 在堆内存中开辟空间，分配地址。
4.堆内存 并在对象空间中，对对象的属性进行默认初始化
5.栈内存 子类构造方法进栈
6.堆内存 显示初始化父类的属性
7.栈内存 父类构造方法进栈，执行完毕出栈
8.堆内存 显示初始化子类的属性
9.栈内存 初始化完毕后，将堆内存中的地址值赋给引用变量。子类构造函数出栈。

多态

Java引用变量有两个类型：编译时类型和运行时类型。编译时类型由声明该变量时使用的类型决定，运行时类型由实际赋给给变量的对象决定。
  若编译时类型和运行时类型不一致，就出现多态。
  子类可看成特殊的父类，所有父类类型的引用可以指向子类的对象：向上转型（upcasting）
      向上转型时：（相当于c++动态联编）
          成员方法：
              编译时：要查看引用变量所属的类中是否有所调用的方法（父类的方法）
              运行时：调用实际对象所属的类中的重写方法（子类中重写或继承的方法）
              （不能调用子类中特有的方法）
          成员变量：
              不具备多态性，只看引用变量所属的类

== 与 equals

1.== 比较对象时 只有指向同一对象地址时才返回true
2.使用 == 时，符号两边的数据类型必须兼容
3.equals只比较引用时作用与 == 相同
4.于某些特殊的类，如String，File，Date，使用 == 比较的是对象的地址，equals比较的是内容
